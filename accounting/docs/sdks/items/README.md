## create

The *Create item* endpoint creates a new [item](https://docs.codat.io/accounting-api#/schemas/Item) for a given company's connection.

[Items](https://docs.codat.io/accounting-api#/schemas/Item) allow your customers to save and track details of the products and services that they buy and sell.

**Integration-specific behaviour**

Required data may vary by integration. To see what data to post, first call [Get create item model](https://docs.codat.io/accounting-api#/operations/get-create-items-model).

Check out our [coverage explorer](https://knowledge.codat.io/supported-features/accounting?view=tab-by-data-type&dataType=items) for integrations that support creating an account.


### Example Usage

```typescript
import { Accounting } from "@speakeasy-sdks/accounting";
import { ItemStatus, ItemType } from "@speakeasy-sdks/accounting/sdk/models/shared";

async function run() {
  const sdk = new Accounting({
    authHeader: "Basic BASE_64_ENCODED(API_KEY)",
  });

  const companyId = "8a210b68-6988-11ed-a1eb-0242ac120002";
  const connectionId = "2e9d2c44-f675-40ba-8049-353bfcb5e171";
  const item = {
    billItem: {
      accountRef: {},
      taxRateRef: {},
    },
    invoiceItem: {
      accountRef: {},
      taxRateRef: {},
    },
    isBillItem: false,
    isInvoiceItem: false,
    itemStatus: ItemStatus.Active,
    metadata: {},
    modifiedDate: "2022-10-23T00:00:00Z",
    sourceModifiedDate: "2022-10-23T00:00:00Z",
    supplementalData: {
      content: {
        "key": {
          "key": "<value>",
        },
      },
    },
    type: ItemType.Service,
  };
  const timeoutInMinutes = 417458;
  
  const result = await sdk.items.create(companyId, connectionId, item, timeoutInMinutes);

  // Handle the result
  console.log(result)
}

run();
```
